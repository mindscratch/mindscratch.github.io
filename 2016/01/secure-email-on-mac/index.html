Secure Email on Mac OS X

This will be quick. I use GMail but it lacks the native ability to sign and encrypt email. I did some looking around for desktop email clients for Mac OS X that support S/MIME.

I ended up going with Airmail 2 because it has S/MIME support and it supports Markdown which I like to use.

S/MIME support is provided by a plugin. Follow the instructions after going to the link, you’ll ultimately download a zip file.

Open Airmail, click on the Window menu and select Show Plugins Folder.... Extract the S/MIME plugin zip file and copy the .bundle file into the plugins folder.

Click on the Window menu and choose Plugins and click Reload Plugins.

You should be all set to send signed and encrypted email.

note: Comodo offers a free email certificate.
